How does it works ?

Search for keywords and see (try "christmas" or "trump" for example). Each point is a tweet, each tweet is ranked as positive or negative. VADER is an algorithm that computes the sentiment expressed in a sentence (positive, neutral, negative). Each category is a number between 0 and 1 (1 being the maximum intensity). When doing a search keep in mind that the keywords you use have a polarity themselves. It is without surprises that searching for "hate" will return a majority of negative results (pay attention to the positive tweets in that case!). Choosing the scope 'gps' option will only show precisely geolocated tweets; when using 'city' scope, the tweet was indeed posted within the city boundary but the point accuracy is not guaranteed.
